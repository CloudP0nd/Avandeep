# Vampire Survivors Clone

リッチなグラフィックを持つヴァンパイアサバイバー系のゲームです。

## 特徴

- **自動射撃**: 最も近い敵に自動的に弾丸を発射
- **レベルアップシステム**: XP を集めてレベルアップ、ステータスが向上
- **リッチなグラフィック**: 発光エフェクト、パーティクル、グラデーション
- **難易度上昇**: スコアに応じて敵が強力になる

## 操作方法

- **WASD または 矢印キー**: プレイヤー移動
- **マウス**: カーソル位置の表示（将来的な拡張用）

## 開発環境

モジュール化された設計により、関心の分離を実現：

```
js/
├── config.js      # ゲーム設定
├── input.js       # 入力処理
├── entity.js      # 基本エンティティクラス
├── player.js      # プレイヤークラス
├── enemy.js       # 敵キャラクタークラス
├── bullet.js      # 弾丸クラス
├── xpGem.js       # XP ジェムクラス
├── particle.js    # パーティクルエフェクト
├── renderer.js    # 描画処理
└── main.js        # メインゲームループ
```

## 遊び方

1. ブラウザで `index.html` を開く
2. WASD または矢印キーでプレイヤーを操作
3. 敵を倒して XP を収集
4. レベルアップして強くなる
5. できるだけ長く生き残る！
